Front-end

O desenvolvimento web front-end é o desenvolvimento da interface gráfica do usuário de um site, por meio do uso de HTML , CSS e JavaScript , para que os usuários possam visualizar e interagir com o site

As metas de desempenho se preocupam principalmente com o tempo de renderização, manipulando HTML, CSS e JavaScript para garantir que o site abra rapidamente.
